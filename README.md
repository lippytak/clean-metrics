# Keep it Clean (Metrics)
Code for the public metrics page of: https://github.com/daguar/calfresh-and-so-clean

# Deploy

1. Pull code

2. Install stuff

`bundle install`

3. Set ENV vars

- PLOTLY_USERNAME
- PLOTLY_KEY
- GOOGLE_DRIVE_EMAIL
- GOOGLE_DRIVE_PASSWORD
- CLEAN_METRICS_SPREADSHEET_KEY

4. Start server
`rails s`

That's it! No databases required at this point.
